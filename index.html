FIRSTAlliances.org | PROJECT
Python | Django | Bootstrap | HTML | CSS

A custom-built website that I led development on, which houses a crowd-sourced robotics team directory. Frontend was written from the ground-up - with no starting template - in Bootstrap and REACT. The backend, handling services such as sign up and searching, is powered by Django and a MySQL database.





Stratsy.IO | PROJECT
Unity | C# | Websockets | Django

An online, multi-player strategy game I developed in most of April. The game is made in Unity, utilizing plenty of custom C# code for the game logic and behavior. The site the game is embedded in is a straightforward Django site with a SQLite database for handling user accounts. The game is available online and is free to play - and a mobile version for Android and iOS is coming soon!





FRC Simulator | PROJECT
Java | LibGDX

Robot simulation & driver practice software I wrote for the FIRST Robotics Competition. Java robot code could be simulated on a virtual robot to aid with testing robot code long before a physical robot was available, and the FRC game could be played in full to allow for development and refinement of match strategies. Also doubled as a fun 6 player game!





OpenTale | PROJECT
Java | LUA

A complete recreation of UNDERTALE's battle engine in Java. UNDERTALE is one of my favorite all time games, and I thought it would be a fun experiment to see if I could completely recreate it's battle engine - which was originally made in GameMaker - in Java with no additional libraries or game frameworks. The experiment was a success, and I even added support for writing your own attacks via LUA files.
